Sign Language Animation


Block Diagram
Following process is involved The process involves: i. Audio to Text conversion ii. Mapping the Text to a relevant sign language animation using NLP and Predictive models. iii. Display the animation on screen Audio to Text conversion can be done by using cloud APIs or offline python packages, build using Deep learning technologies. Once the audio is converted into Text, the next step is to map the text to the suitable animation present in the repository. This can be achieved using either NLP techiques, Predictive modeling or getting the similarity score using cosine similarity function. Once the mapping is done, the respective sign animation can be displayed on the screen.
